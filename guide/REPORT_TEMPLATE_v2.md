## 보고서 챕터 구조 (9개 챕터 고정)

### Ⅰ. 기술 개요 및 배경

**목적:** 독자가 이 기술을 처음 접하더라도 이해할 수 있도록 기초를 제공한다.

**포함 내용:**
- 기술의 공식 정의 및 역할 (어떤 시스템/제품에 쓰이는가)
- 핵심 기술 문제 및 이 기술이 등장한 배경
- 기술의 주요 성능 지표 (KPI) 2~4개
- 기술 분류 체계 (대분류 A~D, 기술 특성에 맞게 조정 가능)

**작성 기준:**
- 전문가뿐 아니라 기획·사업 담당자도 읽을 수 있는 수준
- 기술 분류는 이후 특허·R&D 분석의 기준 분류와 반드시 일치해야 함

---

### Ⅱ. 시장 환경 분석

**목적:** 이 기술이 속한 시장의 규모·성장성·경쟁 구도를 파악한다.

**포함 내용:**

1. **시장 규모 및 CAGR 전망** (출처 명시 필수)
   - 복수 출처 비교 시 편차 범위를 명시하고 단일 수치 확정을 피할 것
   - 📊 **Chart 5**: 출처별 시장 규모 추정 및 CAGR 비교 차트

2. **주요국 정책 동향** (한국·미국·중국·유럽·일본)

3. **주요 기업 동향** (시장 참여자 5~10개사)
   - 기업명 / 역할 / 주요 동향

4. **기업 포지셔닝 맵**
   - 📊 **Chart 6**: 주요 기업의 기술 스택 위치 × 공급망 위치 버블 맵
   - X축: 소재·소자 중심 ↔ 시스템·플랫폼 중심
   - Y축: 부품 공급 ↔ 완제품·플랫폼 리더

**데이터 처리 기준:**
- 시장 수치는 반드시 출처 병기 또는 `(E)` 추정 표시
- 시장 규모 수치 간 편차가 클 경우 → 단일값이 아닌 범위(min~max)로 제시
- 기업 포지셔닝은 공개 자료 기반 추정임을 명시

---

### Ⅲ. 특허 정량 분석

**목적:** 이 기술 분야의 특허 출원 현황을 정량적으로 파악한다.

**포함 내용:**

1. **분석 개요**
   - 검색 DB, 검색 기간, 유효 특허 건수
   - 기술 분류별 검색 건수 테이블

2. **전체 출원 동향**
   - 📊 **Chart 4**: 국가별 연도별 출원 추이(꺾은선) + 국가별 누적 비중(수평막대)

3. **주요 출원인 현황** (상위 5~10개사)
   - 출원인 / 건수 동향 / 주요 국가 / 핵심 기술 영역

4. **주요 IPC 분류 동향**
   - 상위 3~5개 IPC 코드 및 의미

**데이터 처리 기준:**
- 실제 특허 CSV 첨부 시 → 직접 집계하여 사용
- 없으면 Claude 추정치 사용, `(E)` 명시

---

### Ⅳ. 핵심 기술 개요

**목적:** 각 기술 영역의 원리와 진화 방향을 한눈에 파악할 수 있도록 요약한다.

**포함 내용:**
- 기술 분류별 핵심 원리 (1~2문장)
- 기술 계보 (진화 방향)
- 응용 분야 또는 상용화 포인트

**작성 기준:**
- 분량 압축 우선 — 심층 내용은 Ⅴ장에서 다루므로 중복 서술 금지
- 5행 내외의 요약 테이블 형태를 기본으로 함

---

### Ⅴ. 주요 R&D 동향 분석

**목적:** 최근 5년간의 학술 연구 흐름과 핵심 성과를 기술 영역별로 심층 분석한다.

**포함 내용:**

1. **R&D 분석 개요**
   - 📊 **Chart 1**: 기술 영역별 연도별 논문 건수 추이 (꺾은선)
   - 핵심 트렌드 3~5개 요약

2. **기술 영역별 R&D 동향** (Ⅰ장 기술 분류 기준, 각 절 구성)
   - 연도별 연구 흐름 변화
   - 주요 연구 그룹 및 성과 테이블 (기관 / 발표 매체 / 핵심 성과)
   - 특허-R&D 시차 분석 코멘트

3. **핵심 키워드 트렌드**
   - 📊 **Chart 2**: 주요 키워드 연도별 출현 빈도 (묶음 막대)
   - 신흥 키워드 vs. 쇠퇴 키워드 구분

4. **R&D-특허 종합 교차 분석**
   - 📊 **Chart 3**: R&D 성장도 × 특허 성숙도 버블 매트릭스
   - 영역별 권장 전략 테이블

**데이터 처리 기준:**
- 실제 논문 Excel 첨부 시 → 직접 집계 사용
- 없으면 Claude 추정치 사용, `(E)` 명시
- 연구 그룹 테이블: 실제 발표 기반만 작성, 허구 인용 금지

---

### Ⅵ. 기술 성장 단계 분석

**목적:** 각 기술 영역의 현재 성장 단계를 판별하여 진입 시점을 판단한다.

**성장 단계 판별 기준:**

| 단계 | 출원 건수 | 출원인 수 | 특징 |
|------|-----------|-----------|------|
| 태동기 | 적고 불규칙 | 소수 선도 기업 | 원천특허 집중 |
| 성장기 | 급격 증가 | 다수 신규 진입 | 응용·개선 특허 폭발 |
| 성숙기 | 안정 또는 감소 | 진입 감소 | 방어·블록킹 특허 집중 |
| 쇠퇴기 | 급감 | 이탈 증가 | 차세대 기술로 전환 |

**포함 내용:**
- 기술 분류별 현재 단계 / R&D 성장도 / 시사점 테이블
- 전체 기술의 종합 단계 판단 및 진입 시점 의견

---

### Ⅶ. 주요 출원인 IP 히스토리 분석

**목적:** 주요 경쟁자의 특허 전략 변화를 시계열로 파악하여 대응 방향을 도출한다.

**포함 내용:**

1. **상위 출원인 3~5개사 IP 히스토리**
   - 시기별(~5년 단위) 핵심 출원 영역 변화
   - R&D 발표 → 특허 전환 패턴 분석

2. **의뢰 기관 보유 IP 현황 및 평가** ← 시나리오별 처리 방식 상이
   - 시나리오 A: 본 절 생략 또는 "해당 없음" 처리
   - 시나리오 B: 의뢰 기관 보유 특허 현황, 권리 범위, 공백 영역 식별
   - 시나리오 C: "의뢰 기관의 현재 IP 현황 확인 필요" 안내 문구 포함

**작성 기준:**
- 실제 특허번호 언급 시 검증된 정보만 사용
- 불확실한 정보는 "(추정)" 명시
- 보유 특허 명세서 PDF 첨부 시 → 확장 모듈 A 자동 연계

---

### Ⅷ. 공백기술 도출 및 IP 포트폴리오 전략

**목적:** 특허 공백 영역을 도출하고 실행 가능한 진입·출원 전략을 제시한다.

**포함 내용:**

1. **공백기술 도출** (3P 분석: 특허·논문·시장 교차)
   - 최소 4개, 최대 8개 도출
   - 공백기술 번호 / 기술명 / 관련 선행특허 / 신규 IP 창출안 / 진입 가능성 ★ 표기

   **시나리오별 작성 방식:**
   - 시나리오 A: 기술 분야 전반의 공백을 산업 일반론으로 서술
   - 시나리오 B: 의뢰 기관 역량과 연결 가능성이 높은 공백기술을 우선 배치하고 연결 근거 명시
   - 시나리오 C: 공백기술 목록 제시 후 "귀사 역량과의 연결 가능성은 별도 확인 필요" 안내

2. **IP 재구성 전략** (4관점)
   - 종류의 확장 (장치·방법·시스템)
   - 국가의 확장 (PCT 전략)
   - 양적 재구성 (포트폴리오 규모)
   - 질적 재구성 (청구항 설계)

**작성 기준:**
- 공백기술은 반드시 Ⅴ장 R&D-특허 교차 분석 결과와 연동
- 실현 가능성 없는 공백기술 나열 금지
- 보유 특허 명세서 첨부 시 → 확장 모듈 A에서 연결 지도 자동 생성

---

### Ⅸ. 결론 및 시사점

**목적:** 보고서 전체를 요약하고 행동 과제를 도출한다.

**포함 내용:**

1. **기술·R&D 동향 요약** (3~5개 핵심 포인트)

2. **우선순위 행동 과제**

   **시나리오별 작성 방식:**
   - 시나리오 A: 기술/시장 관점의 범용 과제 (특정 기업 전제 없음)
   - 시나리오 B: 의뢰 기관의 사업·역량 맥락을 반영한 구체적 과제
   - 시나리오 C: 일반 진입 검토 순서를 기준으로 과제 제시

   | 시점 | 과제명 | 구체 행동 |
   |------|--------|-----------|
   | 즉시 (~6개월) | | |
   | 단기 (1~2년) | | |
   | 중기 (2~4년) | | |
   | 장기 (4년~) | | |

3. **한계 및 유의사항**
   - 데이터 출처 및 추정치 여부 명시
   - FTO 검토 필요 영역 명시
   - 전문 변리사 검토 권고 문구 포함

## ⚙️ 기술 분류 자동 조정 기준

기술 분야에 따라 Ⅰ장의 기술 분류 체계(A~D)는 아래 기준으로 조정한다.

| 기술 분야 유형 | 분류 기준 예시 |
|----------------|----------------|
| 전자·반도체·통신 | 소자 / 회로 / 시스템 / 알고리즘 |
| 소재·화학·배터리 | 소재 조성 / 공정 / 구조 / 응용 |
| 기계·로봇·액추에이터 | 구동 원리 / 소재 / 제어 / 시스템 통합 |
| 바이오·의료기기 | 진단 / 치료 / 소재 / 기기 구조 |
| 에너지·환경 | 발전 / 저장 / 변환 / 시스템 |
| 소프트웨어·AI | 모델 구조 / 학습 방법 / 응용 / 하드웨어 연계 |
